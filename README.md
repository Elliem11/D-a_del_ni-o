# 🎈 Día del Niño — Sonrisas que Iluminan

Sitio web dedicado a la celebración del **Día del Niño en México**, 
desarrollado como práctica de la materia de Desarrollo Web.

## 🌐 Demo en vivo
[[Ver sitio](https://TU-LINK.vercel.app)](https://d-a-del-ni-o.vercel.app/)

## 📋 Descripción
Página web informativa y celebratoria sobre el Día del Niño, que incluye 
información sobre los derechos de la infancia, actividades recreativas 
y una galería de momentos especiales.
